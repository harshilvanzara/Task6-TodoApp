📝 To-Do App (Java Swing)

📌 Description

A simple To-Do List application built using Java Swing.
You can add and delete tasks easily.

🚀 Features

- Add tasks
- Delete selected tasks
- Simple GUI

🛠️ Technologies Used

- Java
- Swing
- VS Code

▶️ How to Run
